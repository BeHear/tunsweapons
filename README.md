# TUNS Weapons

Набор разнообразного оружия для Garry's Mod.

## Установка

1. Скачайте репозиторий
2. Распакуйте папку `tuns` в `garrysmod/addons/`
3. Зайдите в игру — оружие появится в категории "TUNS Weapons"
